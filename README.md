# Next.js + Jest + MUI

Built using the recommended NextJS App Router, get started quickly with a Next.js template using:

- The src/ folder
- App Router
- Jest + User Events
- MUI + MUI Icons
- Prettier for formatting
- Built in CSS modules

## Development

To run the dev server:

```bash
npm run dev
```

Before deploying:

```bash
npm test
npm run format
npm run build
```
